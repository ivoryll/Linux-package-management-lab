# Linux APT Package Management Lab

## Description

This project demonstrates how to use the Advanced Package Tool (APT) in Linux to install, remove, and manage software packages. It includes hands-on practice with Suricata and tcpdump, two common network security tools used by cybersecurity professionals.

---

## 🎯Objectives

- Verify that APT is installed
- Install Suricata
- Verify the installation
- Remove Suricata
- Install tcpdump
- List installed packages
- Reinstall Suricata
- Confirm both applications are installed

---

## 🛠Technologies Used

- Linux (Debian)
- Bash Shell
- APT Package Manager
- sudo
- Suricata
- tcpdump

---

## 💻Commands Used

### Verify APT

```bash
apt
```

### Install Suricata

```bash
sudo apt install suricata
```

### Verify Installation

```bash
suricata
```

### Remove Suricata

```bash
sudo apt remove suricata
```

### Install tcpdump

```bash
sudo apt install tcpdump
```

### List Installed Packages

```bash
apt list --installed
```

## 🛡Security Tools

### Suricata

Suricata is an open-source Intrusion Detection and Prevention System (IDS/IPS) used to detect and monitor suspicious network activity.

### tcpdump

tcpdump is a command-line packet analyzer used to capture and inspect network traffic for troubleshooting and security investigations.
